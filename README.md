# ArmyKnife

Survival kits for Rails.

## Installation

Add this line to your application's Gemfile:

```ruby
group :development do
  gem 'army_knife'
end
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install army_knife

## Usage

```
$ bin/rails g army_knife:install
```

Then start the Wizard.
